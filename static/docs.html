Getting Started

Welcome to CodeEx AI! This documentation will help you get the most out of our AI-powered platform.

Features

  • Multi-provider AI support (OpenAI, Google Gemini, Local AI)
  • Intelligent conversation with context memory
  • Secure authentication with Google OAuth
  • End-to-end encryption for chat messages
  • Web scraping capabilities
  • Advanced AI brain with learning capabilities

API Endpoints

CodeEx AI provides RESTful API endpoints for integration:

  • /api/chat/sessions - Manage chat sessions
  • /api/ai/providers - Get available AI providers
  • /api/ai/feedback - Submit AI feedback
  • /api/storage/analytics - Get storage analytics
